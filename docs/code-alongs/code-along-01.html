Packages

We’ll use the following packages:

  • here() (relative file paths)
  • tidyverse() (data wrangling)
  • gssr() (U.S. General Social Survey data)
  • gssrdoc() (GSS documentation)

Install here() and tidyverse()

Let’s first install the two packages that are available on CRAN.

Copy and paste the following code into your Console pane. Then hit enter.

install.packages("here")

Then, do the same to install the tidyverse package.

install.packages("tidyverse")

Install gssr() and gssrdoc()

Install gssr from ropensci universe. Copy and paste the following code into your Console pane. Then hit enter.

install.packages('gssr', repos = c('https://kjhealy.r-universe.dev', 'https://cloud.r-project.org'))

Also install gssrdoc.Copy and paste the following code into your Console pane. Then hit enter.

install.packages('gssrdoc', repos = c('https://kjhealy.r-universe.dev', 'https://cloud.r-project.org'))

A “tibble” is another name for “tidy dataset,” meaning that the data is organized in structured, clear rows and columns. “(75,699 × 6,867)” means the dataset contains 75,699 rows and 6,867 columns. In social sciences, rows are commonly referred to as “observations” and columns as “variables.” In our case, there are 75,699 observations (e.g., respondents) and 6,867 variables.
